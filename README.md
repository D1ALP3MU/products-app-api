# 🛒 Products App - API

Aplicación web que consume una API de productos y muestra la información dinámicamente.

## 🚀 Tecnologías
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API

## ⚙️ Funcionalidades
- Consumo de API externa
- Renderizado dinámico de productos
- Manejo de errores
- Diseño responsive

## 🌐 API utilizada
Fake Store API

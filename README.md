# 🛒 Ecommerce App - AlejoDev

Aplicación web tipo ecommerce desarrollada con **HTML, CSS y JavaScript**, que consume una API externa y permite gestionar un carrito de compras con persistencia.

---

## 🧠 Descripción

Esta aplicación permite visualizar productos obtenidos desde una API, filtrarlos, buscarlos en tiempo real y gestionarlos dentro de un carrito de compras dinámico.

El proyecto fue desarrollado aplicando buenas prácticas de desarrollo frontend, enfocándose en la experiencia de usuario (UX) y en la organización del código.

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3 (Flexbox, Grid, variables CSS, responsive design)
* JavaScript (ES6+)
* Fetch API
* LocalStorage
* SweetAlert2

---

## ✨ Funcionalidades principales

### 🛍️ Productos

* Consumo de API externa
* Renderizado dinámico
* Diseño responsive
* Hover effects y animaciones

### 🔍 Filtros y búsqueda

* Búsqueda en tiempo real
* Filtrado por categorías
* Feedback visual en inputs

### 🛒 Carrito de compras

* Agregar productos
* Evitar duplicados
* Control de cantidades (➕ / ➖)
* Eliminar productos
* Vaciar carrito
* Cálculo automático del total

### 💾 Persistencia

* Guardado en LocalStorage
* Recuperación del carrito al recargar la página

### 🧩 Modal

* Visualización de detalles del producto
* Modal del carrito interactivo
* Cierre por botón y fuera del modal

### 💳 Checkout simulado

* Validación de carrito vacío
* Confirmación de compra con feedback visual
